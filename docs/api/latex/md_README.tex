A zero-\/trust, end-\/to-\/end encrypted messenger implemented in pure C.

This project focuses on cryptographic correctness, architectural clarity, and long-\/term maintainability. It is designed as a portable core library with untrusted servers and platform-\/specific clients built on top. \hypertarget{md_README_autotoc_md11}{}\doxysection{\texorpdfstring{Project Status}{Project Status}}\label{md_README_autotoc_md11}

Currently implemented:\+
\begin{DoxyItemize}
\item account creation from master key
\item account import from master key
\item cryptographic identity verification
\end{DoxyItemize}

Networking, protocol implementation, and UI are under active design. \hypertarget{md_README_autotoc_md12}{}\doxysection{\texorpdfstring{Philosophy}{Philosophy}}\label{md_README_autotoc_md12}
This messenger is built on a strict {\bfseries{zero-\/trust model}}.
\begin{DoxyItemize}
\item no phone numbers
\item no email addresses
\item no passwords
\item no implicit trust in servers
\end{DoxyItemize}

All trust is derived exclusively from cryptography. The server acts as an untrusted, dumb store for encrypted data and public keys.
